# Captcha Verification Project

## Overview
This is a **Captcha Verification System** built using **HTML, CSS, and JavaScript**.  
The project prevents automated bots from submitting forms by requiring users to correctly enter a CAPTCHA code. It includes a **dynamic captcha generator**, a **reload button**, and **input validation** for enhanced security and usability.


## Features
- **Dynamic Captcha Generation**: Random alphanumeric CAPTCHA is generated each time.  
- **Reload Captcha**: Users can refresh the CAPTCHA for better readability.  
- **Input Validation**: Checks user input against the generated CAPTCHA.  
- **User Feedback**: Displays success or error messages based on input.  
- **Responsive Design**: Fully responsive and works on all devices.  
- **Customizable UI**: Easy to update background images, fonts, and colors.  


## Technologies Used
- **HTML5** – Markup structure  
- **CSS3** – Styling & responsive design  
- **JavaScript (Vanilla)** – Captcha logic and validation  
- **Bootstrap 5** – For buttons and layout  
- **Font Awesome** – Icons for reload button
